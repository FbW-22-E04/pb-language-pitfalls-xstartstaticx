#### 2. Abrunden
```javascript
console.log(0.1 * 0.2);
```
Das obige Beispiel in JavaScript wird gedruckt: `0.020000000000000004`.
Verwende eine Methode, die sicherstellt, dass nur `0.02` gedruckt wird.
